# 📘 Facebook Story Clone

A simple **Facebook Story UI Clone** built using pure **HTML & CSS**, replicating the look and feel of Facebook’s story section along with a responsive navigation bar.

---

## 🚀 Features

* 📱 Responsive Facebook-style navigation bar
* 🔍 Search input UI
* 🎨 Clean dark theme design
* 📸 Story cards with:

  * Background images
  * Profile avatars
  * Username overlay
* ✨ Smooth hover animations on stories
* 🔔 Icon-based navigation (Material Symbols)

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3**
* **Google Material Symbols (Icons)**

## 📸 UI Sections

### 🔹 Navbar

* Facebook logo
* Search bar
* Navigation icons (Home, Video, Marketplace, Groups)
* Action icons (Add, Notifications, Chat)

### 🔹 Story Section

* Story cards with image backgrounds
* Profile picture overlay
* Username displayed at the bottom
* Hover zoom effect

## ▶️ How to Run

1. Download or clone this repository:

   ```bash
   git clone https://github.com/your-username/facebook-story-clone.git
   ```

2. Open the project folder

3. Run the project:

   * Simply open `index.html` in your browser
